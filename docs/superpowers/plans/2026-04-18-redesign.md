- [ ] **Step 1: `function renderCalendar()` 바로 앞에 아래 함수 추가**

```js
function renderStats() {
  const logs = getLogs();
  const monthLogs = logs.filter(l => {
    const [y, m] = l.date.split('-').map(Number);
    return y === viewYear && m === viewMonth + 1;
  });
  const runLogs = monthLogs.filter(l => l.type === 'running');
  const cfLogs = monthLogs.filter(l => l.type === 'crossfit');
  const totalKm = runLogs.reduce((sum, l) => sum + (parseFloat(l.distance) || 0), 0);

  document.getElementById('stats-total').innerHTML = `${monthLogs.length}<span class="stat-unit">회</span>`;
  document.getElementById('stats-running').innerHTML = `${totalKm.toFixed(1)}<span class="stat-unit">km</span>`;
  document.getElementById('stats-crossfit').innerHTML = `${cfLogs.length}<span class="stat-unit">회</span>`;
}
```

- [ ] **Step 3: 파일 맨 아래 초기화 블록 확인**

`renderCalendar();` / `renderLogs();` 두 줄이 있는지 확인. `renderStats()`는 `renderCalendar()` 안에서 호출되므로 별도 추가 불필요.

- 기록 추가/삭제 시 숫자 갱신 확인
